# AI Place 플랫폼 구조 리뷰 및 개선 지시서 (GEO 최적화)

## 📌 목적

본 문서는 AI Place의

1. 업체 상세 페이지
2. 블로그(가이드) 페이지

두 가지 구조를 분석하고,
**GPT / AI 검색 최적화(GEO)** 기준으로 개선 방향을 정의한다.

---

# 1️⃣ 업체 상세 페이지 리뷰

## ✅ 현재 상태 (잘된 점)

### ✔ 구조화 데이터 완성도 높음

* schema.org 적용 (LocalBusiness, OfferCatalog, FAQ 등)
* 검색 엔진 및 AI가 이해하기 쉬운 구조

### ✔ 정보 구성

* 업체명 / 주소 / 전화 / 영업시간
* 서비스 목록 + 가격 범위
* 리뷰 (Google 기반)
* FAQ 포함

👉 결론:
**“기본 정보형 페이지로는 상위 수준”**

---

## ❌ 문제점 (핵심)

### 1. 차별점 없음

* “웹사이트 제작 전문” → 너무 일반적
* 다른 업체와 구분 불가

### 2. 추천 기준 없음

* 어떤 상황에서 이 업체를 선택해야 하는지 없음

### 3. 타겟 사용자 없음

* 누구에게 맞는 업체인지 정의 안됨

---

## 🚀 개선 요구사항

### 🔥 [필수] 추천 대상 섹션 추가

```md
## 추천 대상
- 병원 / 브랜드 / 쇼핑몰 제작 필요한 기업
- 디자인 완성도 중요한 경우
- 전환율 높은 UX가 필요한 사이트
```

---

### 🔥 [필수] 강점(차별점) 섹션 추가

```md
## 핵심 강점
- 브랜드 중심 UI/UX 설계
- 아임웹 + 커스텀 개발 가능
- 평균 제작 기간 2~4주
- 빠른 피드백 및 수정 대응
```

---

### 🔥 [필수] 비교 기준 추가

```md
## 업체 유형 비교

- 템플릿 제작 업체
  → 저렴하지만 디자인 제한

- 개발 중심 업체
  → 기능은 강하지만 디자인 약함

- 디두 (현재 업체)
  → 디자인 + 기능 균형형
```

---

### 🔥 [선택] 추천 문장 추가

```md
👉 이런 경우 추천됩니다:
- "브랜드 이미지가 중요한 홈페이지"
- "깔끔한 디자인과 UX를 동시에 잡고 싶은 경우"
```

---

## 🎯 목표

👉 단순 정보 페이지 ❌
👉 **AI가 추천 가능한 페이지로 전환**

---

# 2️⃣ 블로그 (가이드) 페이지 리뷰

## ✅ 현재 상태 (잘된 점)

### ✔ SEO 구조 완벽

* 지역 + 키워드 포함 제목
* 목차 구성
* 섹션 분리

### ✔ 데이터 기반 콘텐츠

* 평균 평점
* 비용 범위
* 진료 시간
* 통계 데이터 포함

### ✔ 비교형 콘텐츠 포함

* 증상별 추천
* 비용 안내
* 시간 비교

👉 결론:
**“정보형 블로그로는 매우 우수”**

---

## ❌ 문제점 (핵심)

### 1. 추천 이유 없음

* “여드름 → A병원”
* WHY 없음 → AI 추천 불가

---

## 🚀 개선 요구사항

### 🔥 [필수] 추천 이유 추가

```md
### 여드름 / 피부질환
→ 수피부과의원
- 피부질환 중심 진료
- 건강보험 적용 가능
- 기본 치료 비용 합리적

### 리프팅 / 보톡스
→ 닥터에버스 천안점
- 미용 시술 특화
- 야간 진료 가능 (21시)
- 최신 장비 보유
```

---

### 🔥 [필수] 사용자 유형별 추천 추가

```md
## 이런 분들에게 추천합니다

- 비용 중요 + 기본 치료 → 수피부과
- 빠른 시술 + 야간 진료 → 닥터에버스
- 프리미엄 케어 → 샤인빔
```

---

### 🔥 [필수] 업종 구조 정의 추가

```md
## 천안 피부과 유형 정리

1. 질환 치료 중심
→ 비용 저렴, 보험 적용 가능

2. 미용 시술 중심
→ 결과 중심, 가격 다양

3. 프리미엄 클리닉
→ 고가, 고급 서비스
```

---

### 🔥 [선택] 추천 요약 블록

```md
## 한눈에 추천

- 여드름 치료 → 수피부과
- 리프팅 / 시술 → 닥터에버스
- 프리미엄 관리 → 샤인빔
```

---

## 🎯 목표

👉 단순 정보 제공 ❌
👉 **AI가 답변에 그대로 활용하는 구조**

* 데이터: 충분함

* AI 추천 플랫폼
